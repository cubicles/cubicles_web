First, we read our input using readlines() and we proceed to split it before saving it on the list[]. For the first part, we loop between 0 and the lenght of the input, that means we go line by line. To avoid what will happen when the limit exceeds the width of the string, we use mod function (%) between a pointer that will be the 3 step one. Evaluator in this case is i,k, i step increases one by one and k increases 3 by 3. If evaluator equals to #, the count adds up and the function returns the sum.

For the second part, we slightly modify the function so it takes right and down for parameters. We obtain the required cases and multiply them.
